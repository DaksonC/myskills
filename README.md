# myskills
Projeto do curso React-Native da Rocketseat do Ignite Chapter I

## Instalação e execução do projeto  
-          git clone https://github.com/DaksonC/myskills.git
-          cd myskills
  
  Ambiente Windows
-          yarn android
ou
-          react-native run-android
  
  Ambiente iOS  
-          yarn ios
ou
-          react-native run-ios
  
  Ambiente Linux 
-          yarn start
ou
-          react-native start
depois
-          yarn android
ou
-          react-native run-android        

## Tecnologias

- React Native
- TypeScript
- Android Studio
